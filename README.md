# 🤖 我的 AI 学习路径

> 记录我的人工智能学习旅程，持续更新中...

## 📚 学习路线总览

```
阶段一：数学与编程基础
    ├── Python 编程
    ├── 线性代数
    ├── 概率与统计
    └── 微积分基础

阶段二：机器学习基础
    ├── 监督学习（回归、分类）
    ├── 无监督学习（聚类、降维）
    ├── 模型评估与调优
    └── 经典算法实践

阶段三：深度学习
    ├── 神经网络基础
    ├── CNN（卷积神经网络）
    ├── RNN / LSTM
    └── 框架实战（PyTorch / TensorFlow）

阶段四：自然语言处理（NLP）
    ├── 文本预处理
    ├── 词嵌入（Word2Vec, GloVe）
    ├── Transformer 架构
    └── 大语言模型（LLM）

阶段五：前沿与实战
    ├── Prompt Engineering
    ├── RAG（检索增强生成）
    ├── AI Agent
    └── 项目实战
```
